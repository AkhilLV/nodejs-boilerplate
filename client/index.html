Recursion

Calling a function within the function. Can be used when the problem is in the form of a piecewise function

1. Factorials

          
const factorial = n => {
  if (n >= 1) {
    return n * factorial(n - 1)
  }
  return 1
}

2. Fibonacci

          
const fibonacci = (n) => {
  if (n >= 3) { return fibonacci(n - 1) + fibonacci(n - 2) }
  return 1
}
